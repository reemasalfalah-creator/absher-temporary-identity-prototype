# Temporary Digital Identity Prototype (Absher Hackathon) 

## Overview
This project is a prototype developed during the Absher Hackathon. It proposes a new feature that allows citizens to request a temporary digital identity when they prefer not to share their real national identity.

The concept aims to enhance user privacy while maintaining identity verification through official platforms.

## Idea
- Provide users with a temporary identity for limited-use scenarios
- Protect personal data and privacy
- Simulate integration with Absher services (prototype only)

## Scope
- Front-end prototype only
- Not connected to real government systems
- Designed to demonstrate user flow and concept feasibility

## Technologies Used
- HTML
- CSS

## Use Case
- Citizens who want to protect their identity while accessing services
- Privacy-sensitive digital interactions
